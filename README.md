## 프로젝트 명 : AutoStudy


### 서비스 목표
- 내가 알고 있는 프로그래밍 언어로 다른 언어를 빠르게 학습합니다


### 기능
- 생성형 AI로 문제를 생성하고 채점 및 피드백을 제공받습니다
- 질문 사항을 생성형 AI와 채팅을 통해 질의 응답을 받습니다
- 회원은 자신이 푼 문제에 대한 히스토리 목록을 조회하고, 상세보기를 할 수 있습니다.
